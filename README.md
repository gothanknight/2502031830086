## Project: AI Recipe Generator (Rule-Based)

A fully functional frontend website that simulates an AI-based recipe generator using rule-based logic in JavaScript. Based on user inputs (ingredients, cuisine, preferences), the system suggests recipes dynamically — no backend, no ML, no APIs.
