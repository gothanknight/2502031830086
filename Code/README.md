# Website Code Folder

This folder contains the complete frontend website for the **AI Recipe Generator** project.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript (ES6+)
- jQuery 3.7

## Pages
| File | Description |
|---|---|
| index.html | Home Page |
| login.html | Login / Signup Page |
| ingredients.html | Ingredient Selection |
| cuisine.html | Cuisine Selection |
| preferences.html | User Preferences |
| results.html | Recipe Results |
| recipe-detail.html | Recipe Detail View |
| favorites.html | Saved Favorites |
| about.html | About Page |
| contact.html | Contact Page |

## Assets
- `assets/css/style.css` — Custom stylesheet
- `assets/js/recipes-data.js` — Recipe database + AI matching engine
- `assets/js/app.js` — Application logic
- `assets/js/components.js` — Reusable navbar/footer components
