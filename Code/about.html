Project Overview

RecipeAI is a fully frontend-based web application that simulates artificial intelligence using rule-based JavaScript logic. By selecting ingredients, cuisine types, and dietary preferences, users receive instant recipe recommendations — all without any backend server, external API, or machine learning framework.

The project demonstrates how intelligent behavior can be simulated through well-structured conditional logic, scoring algorithms, and data-driven programming. The "AI" scores each recipe based on ingredient overlap, cuisine match, dietary compliance, and spice preference — producing ranked results that feel genuinely personalized.

🧠 How the Rule-Based AI Works

📥
1. Input Collection

User selects ingredients, cuisine, diet type, spice level, and cooking time through a guided multi-step flow.

⚖️
2. Scoring Engine

Each recipe gets scored: +40 pts per matching ingredient, +30 for cuisine, +20 for diet match, +10 for spice level.

📊
3. Ranking

Recipes are sorted by total score descending. A "match percentage" is calculated for each result, displayed as a badge.

✅
4. Display

Top-matching recipes are rendered dynamically with full details including ingredients, steps, and nutrition info.
