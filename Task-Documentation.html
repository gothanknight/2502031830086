Task 3 — Develop Website (HTML, CSS, JS, jQuery, Bootstrap)

Step 1: Created the project folder structure with separate folders for HTML pages and assets:

Code/ → assets/css/, assets/js/, *.html

Step 2: Built a custom stylesheet assets/css/style.css using CSS3 variables, animations, flexbox layouts, and responsive media queries — matching the Figma design system exactly.

Step 3: Created the Rule-Based AI engine in assets/js/recipes-data.js:

  • Defined RECIPES_DB — array of 15+ recipe objects with cuisine, diet, ingredients, spice, steps
  • Built matchRecipes() — scoring function: +40 pts/ingredient match, +30 cuisine, +20 diet, +10 spice
  • Returns sorted results with a match percentage badge

Step 4: Built reusable components in assets/js/components.js using jQuery — Navbar, Footer, Step Progress Bar, and Page Header injected via data-component attributes across all pages.

Step 5: Implemented all 10 HTML pages using Bootstrap 5.3 grid, with jQuery event handling for interactivity. Multi-step flow stores state in sessionStorage. Favorites use localStorage.
